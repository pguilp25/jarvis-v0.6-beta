WHEN DESIGNING UI — COLOR AND VISUAL DESIGN GUIDELINES

DO NOT use random colors. Every color choice must be intentional.
DO NOT use the default "AI look" — generic blue/purple gradients, neon accents,
dark backgrounds with bright borders. These look like every other AI project.

COLOR PRINCIPLES:
- Pick a SMALL palette: 2-3 main colors + neutrals. Not 8 different colors.
- Colors must WORK TOGETHER. Use color theory:
  * Complementary: opposite on the wheel (high contrast, use sparingly)
  * Analogous: neighbors on the wheel (harmonious, natural feel)
  * Monochromatic: one hue, different shades (clean, professional)
  * Split-complementary: one color + two neighbors of its complement (balanced)
- Background and text need HIGH CONTRAST for readability (4.5:1 minimum)
- Accent color should be used SPARINGLY — for buttons, links, active states.
  Not for backgrounds, not for borders, not for everything.
- Neutral colors (grays, off-whites, muted tones) should do most of the work.
  The accent color draws attention to what matters.

WHAT MAKES A UI LOOK GOOD (not generic):
- Consistent spacing — use a spacing scale (4px, 8px, 12px, 16px, 24px, 32px)
- Typography hierarchy — clear difference between headings, body, captions.
  Use font weight and size, not just color, to create hierarchy.
- Subtle shadows and borders — not hard 1px borders everywhere.
  Use soft shadows (0 2px 8px rgba(0,0,0,0.1)) for depth.
- Rounded corners should be consistent — pick one radius and use it everywhere.
  Don't mix 4px, 8px, 12px, 20px on different elements.
- Whitespace is a design element — don't cram things together.
- Icons should match the style — don't mix filled and outlined icons.
- Hover states and transitions (0.15s-0.2s ease) make it feel polished.

COMMON MISTAKES:
- Too many colors → looks like a toy
- No contrast between elements → everything blends together
- Borders on everything → looks like a spreadsheet
- Default browser form styles → looks unfinished
- Different border-radius values → looks inconsistent
- Neon colors on dark background → looks like a terminal from 1985
- Gradients everywhere → looks like a template

WHEN CHOOSING COLORS FOR A SPECIFIC PROJECT:
Think about the MOOD the project wants to convey:
- Professional/business → navy, slate, white, subtle blue accent
- Creative/fun → warm colors, playful combinations, more saturation
- Technical/developer → dark neutrals, monospace hints, green or cyan accent
- Clean/minimal → lots of white, one accent color, thin lines
- Warm/friendly → earth tones, rounded shapes, warm grays
